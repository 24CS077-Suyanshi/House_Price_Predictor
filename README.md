# 🏠 House Price Predictor

## 📌 Overview

The House Price Predictor is a Machine Learning project developed during my **6-Week AI & Machine Learning Internship at Synkoc IT Services Private Limited, Bengaluru**.

The project predicts house prices using the California Housing dataset. Different regression models were trained and compared to build an Automated Valuation Model (AVM) for real estate price prediction.

---

## ❓ Problem Statement

Estimating house prices accurately is difficult because prices depend on multiple factors such as location, income, population, and housing characteristics. Real estate companies need a system that can predict property prices quickly and accurately.

---

## 💡 Solution

Built an **Automated Valuation Model (AVM)** using Machine Learning techniques that:

* Analyzes housing data
* Creates new features from the dataset
* Trains multiple regression models
* Compares model performance
* Predicts house prices automatically

## 📊 Dataset

**Dataset:** California Housing Dataset

**Source:** `sklearn.datasets.fetch_california_housing()`

The dataset contains information such as:

* Median Income
* House Age
* Average Rooms
* Average Bedrooms
* Population
* Average Occupancy
* Latitude
* Longitude

**Note:** The dataset is loaded directly from Scikit-learn and is not included in this repository.
